OverTheWire Bandit Level 11 Write‑up

Updated: 19/04/2024
| 1 min read

The content of the file this time is encoded with rot13, which is a simple letter substitution cipher that replaces each letter with the 13th letter after it. 

				
					bandit11@bandit:~$ cat data.txt
					Gur cnffjbeq vf WIAOOSFzMjXXBC0KoSKBbJ8puQm5lIEi

Various methods can be used to solve this challenge, including online rot13 decoders and a command-line utility called rot13. However, for educational purposes we will use the tr command, which is used to translate or delete text. tr takes two sets as arguments, the first one indicates the selected characters and the second the translation operation. Since the english alphabet has 26 characters, rot13 happens to be its own reverse, thus applying it again will give us the solution. The tr command in full form is: 

				
					bandit11@bandit:~$ cat data.txt | tr "[abcdefghijklmnopqrstuvwxyz][ABCDEFGHIJKLMNOPQRSTUVWXYZ]" "[nopqrstuvwxyzabcdefghijklm][NOPQRSTUVWXYZABCDEFGHIJKLM]"
However we can shorten the arguments like so: 
				
					bandit11@bandit:~$ cat data.txt | tr "[a-z][A-Z]" "[n-za-m][N-ZA-M]"
					The password is JVNBBFSmZwKKOP0XbFXOoW8chDz5yVRv
Explanation of the command:
  • "[a-z][A-Z]", selects both upper and lowercase letters at the same time
  • "[n-za-m][N-ZA-M]", maps each letter 13 places forward

And that's the password for the next level!
