OverTheWire Bandit Level 3 Write-up

Updated: 14/04/2024
| 1 min read

We already know that the password for the next level is stored in a hidden file inside the inhere directory. So let's cd into that directory first. 

				
					bandit3@bandit:~$ ls
					inhere
					bandit3@bandit:~$ cd inhere/

Terminal commands have a lot of additional options available alongside their default usage. These options are called "flags" or "switches". When described by a single letter they can be addressed with a dash, and if the option is an entire word we can use a double dash. For example, to make sure we can view hidden files with the ls command we have to add the -a flag. The -l flag stands for long listing and gives us additional information, and -h makes the file sizes human readable. 

				
					bandit3@bandit:~/inhere$ ls -lah
					total 12K
					drwxr-xr-x 2 root    root    4.0K   06:19 .
					drwxr-xr-x 3 root    root    4.0K   06:19 ..
					-rw-r----- 1 bandit4 bandit3   33   06:19 .hidden

And there it is!

note: hidden files in Linux always start with a dot
				
					bandit3@bandit:~/inhere$ cat .hidden
					2EW7BBsr6aMMoJ2HjW067dm8EgX26xNe

Level 3 done!
