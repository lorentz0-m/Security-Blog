OverTheWire Bandit Level 0 Write‑up

Updated: 14/04/2024
| 2 min read

The main goal of this level is to connect to the overthewire server via SSH.

SSH is a utility used for creating a secure shell on a server or another computer through a network. Traditionally Telnet was used, but since it didn't provide any form of encryption and always sent data as plain text it got deprecated as the internet grew in size.

You can connect to each level by executing:

				
					$ ssh banditX@bandit.labs.overthewire.org -p 2220
Explanation of the command:
  • X is the number corresponding to each level
  • banditX is the username you are trying to login as
  • bandit.labs.overthewire.org is the domain name of the bandit wargame
  • the -p flag specifies the port number you want to connect to, in our case its 2220 since that's the port for all of the bandit challenges

For example in this level we have:

				
					$ ssh bandit0@bandit.labs.overthewire.org -p 2220

After establishing a connection you will be prompted to enter a password. For this level it's simply bandit0. If you want to input the password in advance you can also use sshpass and reference the password from a file. 

				
					$ sshpass -f <file name> ssh bandit0@bandit.labs.overthewire.org -p 2220
tip: to read the manual of a command reference its manpage by executing $ man <command name>, or $ help <command name> if its a shell built in like cd (change directory). In case the manual is split into different sections you can view each one by executing $ man <section number> <command name>.

Now that you've logged in a shell you can use ls to list the content of your home directory (indicated with a tilde ~) 

				
					bandit0@bandit:~$ ls
					readme

A readme file appears to be stored inside the home dir, by using the cat command (short for concatenate) you can view its content inside the terminal 

				
					bandit0@bandit:~$ cat readme
					NH2SXQwcBdpmTEzi3bvBHMM9H66vVXjL

Which is the password for level 1!
