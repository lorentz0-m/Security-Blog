OverTheWire Bandit Level 18 Write‑up

Updated: 30/04/2024
| 1 min read

The password for level 19 is stored in a readme file inside ~, the same way it was in level 0. However, the .bashrc script of bandit18 has been modified so it logs us out every time we log in with SSH. To go around this issue we can use the "command" option of the ssh utility, which speficies a command to be executed on the remote host. In this way we can cat the readme file in our console without having to log in. 

				
					$ ssh bandit18@bandit.labs.overthewire.org -p 2220 command cat readme

								_ 					   _ _ _
								| |__   __ _ _ __   __| (_) |_
								| '_ \ / _` | '_ \ / _` | | __|
								| |_) | (_| | | | | (_| | | |_
								|_.__/ \__,_|_| |_|\__,_|_|\__|

							This is an OverTheWire game server.
						More information on http://www.overthewire.org/wargames

					bandit18@bandit.labs.overthewire.org's password:
					awhqfNnAbc1naukrpqDYcF95h7HoMTrC

And there we go!

Bonus: If you are curious about how the .bashrc script manages to log out users, you can view the .bashrc file itself the same way we did with readme.

note: the .bashrc script is located inside the home directory and it is the script that bash runs every time a terminal opens. 
				
					# ~/.bashrc: executed by bash(1) for non-login shells.
					# ---snip---
					echo 'Byebye !'
					exit 0

Interesting, turns out it was the built-in exit command.
