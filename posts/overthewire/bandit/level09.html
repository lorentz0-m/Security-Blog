OverTheWire Bandit Level 9 Write‑up

Updated: 29/11/2024
| 0 min read

This time the password is one of the few human-readable strings, which means we will resort to the strings command again! We know that it is preceded by several "=" characters, so we will assume it has at least two equal signs. Applying the previous we get: 

				
					bandit9@bandit:~$ strings data.txt | grep "=="
					}========== the
					3JprD========== passwordi
					~fDV3========== is
					D9========== FGUW5ilLVJrxX9kMYMmlN4MgbpfMiqey

We can easily make out the password for level 10 to be in the last line!
