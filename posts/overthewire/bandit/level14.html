OverTheWire Bandit Level 14 Write‑up

Updated: 25/04/2024
| 0.5 min read

The hint for getting the password this time is to submit the credentials of the current level to port 30000 on localhost. First of all, when you are already connected to a level, you have access to its password from a file named after your username under /etc/bandit_pass. In this case we have: 

				
					bandit14@bandit:~$ cat /etc/bandit_pass/bandit14
					fGrHPx402xGC7U7rXKDaxiWFTOiF0ENq

Now we need a utility to create a connection with our machine (or localhost, pretty much the overthewire server we connected to through SSH) on port 30000 so we can submit the password. The right tool for the job is netcat, also known as "nc", which can read and write data across network connections. We can pipe the content of bandit14 to nc like so: 

				
					bandit14@bandit:~$ cat /etc/bandit_pass/bandit14 | nc localhost 30000
					Correct!
					jN2kgmIXJ6fShzhT2avhotn4Zcka6tnt
Explanation of the command:
  • localhost refers to the target machine
  • 30000 is the port number we want to connect to, notice the -p flag isn't required in this case like it is with ssh, for more details check out the manpage of netcat

Level 14 done!
