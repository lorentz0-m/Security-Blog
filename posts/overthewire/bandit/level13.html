OverTheWire Bandit Level 13 Write‑up

Updated: 29/11/2024
| 0.5 min read

This time we get a private SSH key that can be used to log into the next level

				
					bandit13@bandit:~$ ls
					sshkey.private
					bandit13@bandit:~$ cat sshkey.private
					-----BEGIN RSA PRIVATE KEY-----
					MIIEpAIBAAKCAQEAxkkOE83W2cOT7IWhFc9aPaaQmQDdgzuXCv+ppZHa++buSkN+
					gg0tcr7Fw8NLGa5+Uzec2rEg0WmeevB13AIoYp0MZyETq46t+jk9puNwZwIt9XgB
											---snip---
					kAWpXbv5tbkkzbS0eaLPTKgLzavXtQoTtKwrjpolHKIHUz6Wu+n4abfAIRFubOdN
					/+aLoRQ0yBDRbdXMsZN/jvY44eM+xRLdRVyMmdPtP8belRi2E2aEzA==
					-----END RSA PRIVATE KEY-----

To use the sshkey.private file we have to add the -i flag in the ssh command, which specifies the identity file we want to use. 

				
					bandit13@bandit:~$ ssh -i sshkey.private bandit14@bandit.labs.overthewire.org -p 2220

After executing the command make sure to confirm the SSH connection, and that was it! Alternatively you can use localhost instead of the overthewire domain name to connect: 

				
					bandit13@bandit:~$ ssh -i sshkey.private bandit14@localhost -p 2220
Explanation of the command:
  • the "localhost" hostname refers to the current computer used to access it, the IP of localhost is 127.0.0.1 which is the standard address for IPv4 loopback traffic, meaning network traffic that is sent to the same machine that you are using
