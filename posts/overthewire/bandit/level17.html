OverTheWire Bandit Level 17 Write‑up

Updated: 29/04/2024
| 0 min read

We have two files our home directory, passwords.old and passwords.new

				
					bandit17@bandit:~$ ls
					passwords.new  passwords.old

The hint is that the password for the next level is the only line that has been changed between the old and new password files. We can easily compare the two files line by line to determine any differences by using the diff command like so: 

				
					bandit17@bandit:~$ diff passwords.old passwords.new
					42c42
					< p6ggwdNHncnmCNxuAt0KtKVq185ZU7AW
					---
					> hga5tuuCLF6fFzUpnagiMN8ssu9LFrdg

The bottom line has been replaced in the second file so it's the password we were looking for!

note: if you prefer a colored output (for distinguishing differences more easily) you can use the colordiff command, which has the same functionality but adds color to stdout! However it doesn't appear to be installed on overthewire.
