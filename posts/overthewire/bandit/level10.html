OverTheWire Bandit Level 10 Write‑up

Updated: 29/11/2024
| 0 min read

The data.txt file now is base64 encoded, the command which will help us is base64, which decodes base64 data and vice versa, make sure to include the -d flag so decoding is specified. 

				
					bandit10@bandit:~$ base64 -d data.txt
					The password is dtR173fZKb0RRsDFSGsg2RWnpNVj3qRr
