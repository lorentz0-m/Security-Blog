OverTheWire Bandit Level 7 Write‑up

Updated: 29/11/2024
| 0.5 min read

A file called data.txt appears to be inside our home directory.

				
					bandit7@bandit:~$ ls
					data.txt

However, if we try to cat the file inside the console we can quickly see the terminal fill up with garbage data: 

				
					bandit7@bandit:~$ cat data.txt
					gallop	hu3ZhCrGRvfaO5jsY6ttvApzVCA2Hjvs
					Aurelia's	ikl4F3cK5m6Cl6HAxva6zUAVJhI2Cvc6
					stoicism	JiW9ts44udf20bJHe8H5dS1c99Muwz42
					embodies	vWheZcAsQHZNnerI3ViW8wqOKIx0kbgR
					Plato	dW2U8E5FfuAvNLdGDupP8GAxr922ZV0x
					cultivation	A90E75jvWbEKrijFxM4GxqHEw8c8U2Bf
					stable	omR4PHolFwbI0IEJsanveA21yWvFy8a7
					---snip---

Since we know that the password for the next level is next to the word millionth, we can use the grep command (first seen back in level 5) to easily locate it: 

				
					bandit7@bandit:~$ grep "millionth" data.txt
					millionth	dfwvzFQi4mU0wfNbFOe9RoWskMLg7eEc

Onto level 8!
